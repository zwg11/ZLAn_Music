# ZLAn_Music

vue项目仿网易云音乐

 **后端接口** 

本项目后端接口是Github中Binaryify的开源项目（项目地址：[https://github.com/Binaryify/NeteaseCloudMusicApi](http://)），接口文档直接在百度搜索“网易云音乐API”作者是Binaryify。
